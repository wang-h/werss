# 自动错误检测与修复系统

## 功能说明

这个系统可以自动检测代码中的常见错误，并提供自动修复功能。

## 使用方法

### 1. 在浏览器中查看错误面板

启动开发服务器后，访问：
```
http://localhost:3000/error-dashboard.html
```

### 2. 运行测试

在终端运行：
```bash
pnpm run test:auto-fix
```

这个命令会：
1. 运行 Playwright 测试检测错误
2. 生成错误报告
3. 自动应用可修复的错误

### 3. 单独运行修复

如果只想应用修复（不运行测试）：
```bash
pnpm run fix:apply
```

### 4. 查看测试报告

测试报告会保存在：
```
test-results/auto-fix-report.json
```

浏览器面板会自动读取并显示这个报告。

## 支持的自动修复

目前支持以下类型的自动修复：

1. **属性类型错误** - 如 `strong` 属性问题
   - 自动将 `<Text strong>` 改为使用 CSS 样式

2. **导入错误** - 如 `Header` 组件导入问题
   - 自动将独立的 `Header` 导入改为 `Layout.Header`

3. **缺失 key 属性** - 需要手动修复
   - 会提示需要添加 key 的位置

## 错误面板功能

- ✅ 实时显示错误和警告
- ✅ 显示错误详情和位置
- ✅ 提供修复建议
- ✅ 自动刷新（每10秒）
- ✅ 错误分类统计

## 注意事项

- 自动修复会直接修改源代码文件
- 建议在修复前先提交代码或创建备份
- 某些复杂错误需要手动修复
- 修复后建议重新运行测试验证
